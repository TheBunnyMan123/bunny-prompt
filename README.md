## Bunny Prompt
### Installation
Add the following to your bashrc after installing in your preferred method:
```bash
prompt() {
  PS1=bunny-prompt
}

PROMPT_COMMAND=prompt
```

Check releases for the binary and an arch PKGBUILD
This requires a nerd font
